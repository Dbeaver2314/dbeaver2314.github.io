In my free time I have been studying genetic algorithms and machine learning. This is a project demonstrating a genetic algorithm. The “rockets” goal is to reach the circle at the top of the screen. Each generation has a life span of 8 seconds. At the end of each generation the strongest genes get to reproduce to the next round. In order to better model a real life cycle each generation also includes random “DNA” this can cause huge leaps in development, or hinder development. This project was made using the P5 JavaScript library.
